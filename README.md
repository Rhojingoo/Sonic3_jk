## KEY Manual
→방향키 : 오른쪽이동

←방향키 : 왼쪽이동

SpaceBar : 점프

N : 씬이 넘어 갑니다.

P : 콜라이더 끄기

O : 콜라이더 켜기

K : 링 100개 추가
